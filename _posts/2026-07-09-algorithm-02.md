이번에는 최댓값, subquery에 관한 문제였습니다.
연도별 최대 개체크기와 각 대장균의 개체를 뺀 값을 연도별로 오름차순을 해야하는 문제였습니다.

그래서 저는 최댓값을 구하는 subquery와 각 대장균의 크기를 뺀 값을 구하는 방식으로 query를 작성하였습니다.
```sql
SELECT YEAR(DIFFERENTIATION_DATE) AS YEAR, 
((SELECT MAX(SIZE_OF_COLONY) FROM ECOLI_DATA WHERE YEAR(DIFFERENTIATION_DATE) = YEAR) - SIZE_OF_COLONY) AS YEAR_DEV,
ID FROM ECOLI_DATA 
ORDER BY YEAR, YEAR_DEV ASC
```
